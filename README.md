# 2-Bit Difference Detector

## Overview
This project is a hardware circuit designed to evaluate and output the difference between two independent 2-bit binary inputs. 

**Note:** This was built as a foundational project for a Digital Logic Design course that i took in my first year computer science program. The primary objective was to successfully translate theoretical Boolean algebra, truth tables, and logic minimization into a fully functional physical circuit.

## Hardware Implementation
Rather than utilizing software-based simulations, this system was engineered manually using physical electronic components to process the binary data. The physical architecture relies on:
*   **Breadboards:** Utilized for structural wiring and component layout.
*   **Discrete Logic Gates:** Integrated circuits (ICs) mapped to specific Boolean operations (such as XOR, AND, and OR) to process the input signals.
*   **Power Modules:** Integrated to regulate voltage and maintain stable, consistent high/low logic levels across the circuit.
